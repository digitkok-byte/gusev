# Gusev — Юридическая практика

Лендинг для юридической практики (коммерческая недвижимость, арбитражные споры, сопровождение бизнеса).

## Стек

- Vite + React
- [Motion](https://motion.dev) — scroll- и entry-анимации
- Плоский CSS с CSS Custom Properties (Manrope + Inter)

## Запуск

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # сборка в dist/
npm run preview  # предпросмотр prod-сборки
```

## Структура

- `src/App.jsx` — все секции лендинга
- `src/index.css` — стили и палитра
- `src/assets/portrait.jpg` — фото руководителя практики
